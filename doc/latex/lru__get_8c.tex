\doxysubsection{Detailed Description}
Implementación de la función {\ttfamily get\+\_\+data} para simular el uso de un elemento en la caché LRU. 

Esta función simula la operación de "{}uso"{} o "{}lectura"{} de un elemento en el caché. Si el elemento existe (Cache Hit), se promueve al inicio (MRU) para reflejar que es el dato más recientemente utilizado. Si no existe (Cache Miss), no hace nada.
