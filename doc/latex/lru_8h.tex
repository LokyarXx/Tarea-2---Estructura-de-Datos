\doxysubsection{Detailed Description}
Archivo de cabecera principal para el sistema de Caché LRU. 

Contiene las definiciones de las estructuras de datos (\doxylink{struct_node}{Node} y \doxylink{struct_l_r_u_cache}{LRUCache}) y los prototipos de todas las funciones que componen la API del caché, \mbox{[}cite\+\_\+start\mbox{]}siguiendo la normativa de codificación y modularidad del proyecto. \mbox{[}cite\+: 25, 31\mbox{]}

Implementa la lógica de añadir un dato al caché. 

Si el dato existe, se promueve a MRU. Si no existe, se añade a MRU, \mbox{[}cite\+\_\+start\mbox{]}y se elimina el LRU si la caché está llena. \mbox{[}cite\+: 18, 19\mbox{]}

Implementa la lógica de añadir un dato al caché.

Los casos manejados son\+:
\begin{DoxyEnumerate}
\item {\bfseries{Cache Hit\+:}} Si el dato existe, se remueve de su posición y se inserta al inicio (MRU).
\item {\bfseries{Cache Miss (Lleno)\+:}} Si el dato no existe y la caché está llena, se elimina el nodo LRU (tail), y luego se añade el nuevo nodo al inicio (MRU).
\item {\bfseries{Cache Miss (Hay espacio)\+:}} Si el dato no existe y hay espacio, se añade directamente al inicio (MRU).
\end{DoxyEnumerate}
