\doxysubsection{Detailed Description}
Implementación de la función para mostrar el contenido completo de la caché LRU. 

Esta función recorre la lista doblemente enlazada desde el head (MRU) hasta el tail (LRU) e imprime los datos de cada nodo, permitiendo al usuario visualizar el estado actual y el orden de prioridad del caché.
