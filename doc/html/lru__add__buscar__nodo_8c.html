Descripción detallada

Implementación de la función auxiliar para buscar un nodo dentro de la caché.

Esta función recorre la lista doblemente enlazada desde el MRU (head) para encontrar la primera ocurrencia de un dato específico. Es utilizada por las funciones de manipulación del caché como add_data() y get_data().

Node * buscar_nodo ( LRUCache * cache,
char data )

Busca un nodo específico en la caché por su dato.

Recorre la lista desde el head (MRU) hasta el tail (LRU). Si el dato es encontrado (Cache Hit), retorna un puntero al nodo. Si no se encuentra (Cache Miss), retorna NULL. Esta es una operación de solo lectura y no modifica la prioridad del dato.

Parámetros
cachePuntero a la estructura LRUCache.
dataEl dato (carácter) a buscar.
Devuelve
Node* Puntero al nodo encontrado, o NULL si el dato no está presente.
