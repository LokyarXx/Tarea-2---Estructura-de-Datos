Descripción detallada

Implementación de la función free_cache para liberar toda la memoria dinámica de la caché LRU.

Esta función es crítica para evitar fugas de memoria (memory leaks). Recorre la lista doblemente enlazada, liberando nodo por nodo, y finalmente libera la estructura principal LRUCache.

void free_cache ( LRUCache * cache)

Libera toda la memoria dinámica asociada a la caché LRU.

Libera toda la memoria dinámica asociada al caché.

La liberación se realiza en dos fases:

  1. Recorre la lista enlazada (desde el head) y libera cada nodo individualmente.
  2. Finalmente, libera la estructura LRUCache principal. Se incluye una verificación para asegurar que la función pueda ser llamada incluso si la caché nunca fue creada.
