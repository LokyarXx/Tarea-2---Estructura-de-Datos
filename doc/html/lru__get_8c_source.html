5 * Esta función simula la operación de "uso" o "lectura" de un elemento en el caché.
6 * Si el elemento existe (Cache Hit), se promueve al inicio (MRU) para reflejar
7 * que es el dato más recientemente utilizado. Si no existe (Cache Miss), no hace nada.

9 * * @authors

19/**
20 * @brief Simula el uso de un dato en la caché LRU, promoviéndolo a MRU si existe.
21 *
22 * Busca el dato en el caché.
23 * - **Cache Hit:** Si el nodo es encontrado, se remueve de su posición actual
24 * y se añade al inicio de la lista (MRU).
25 * - **Cache Miss:** Si el nodo no es encontrado, no se realiza ninguna modificación.
26 *
27 * @param cache Puntero a la estructura LRUCache.
28 * @param data El carácter (letra) a buscar y usar.
29 * @return void No retorna valor.
30 */
31
32void get_data(LRUCache* cache, char data)
33{ //Comprobación de errores
34 if (cache == NULL || cache->capacity == 0)
35 {
36 printf("Error: El cache no ha sido inicializado.\n");
37 return;
38 }
39 //Revisa si existe el nodo en el cache
40 Node* existente = buscar_nodo(cache, data);
41 if (existente != NULL) //Si existe el nodo:
42 {
43 remove_node(cache, existente); //Lo quita de su posición actual
44 add_inicio(cache, existente); //Lo añade al inicio (MRU)
45 printf("\nElemento %c encontrado y movido a inicio.\n", data);
46 }
47 else
48 { //Si el nodo no existe:
49 printf("\nElemento %c no encontrado en el cache.\n", data);
50 }
51}
